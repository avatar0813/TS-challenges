## 项目目标

该项目用于学习`Typescript`类型变成，题库来源于 github仓库 [type-challenges](https://github.com/type-challenges/type-challenges)

目前只解决实现题库中的初级、中级题，很多高级题过于复杂，暂缓高级题的探讨；将认为有比较记录的解题思路以及一开始的错误解答方式记录与每个题的`template.ts`中;
